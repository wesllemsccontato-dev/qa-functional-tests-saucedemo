#  Teste Exploratório

##  Objetivo
Explorar comportamentos inesperados na funcionalidade de login.

## Cenários testados

- Atualizar página após login
- Inserir espaços nos campos
- Clicar múltiplas vezes no botão
- Testar copy/paste
- Testar campos vazios

## Observações

- Sistema não mantém sessão após refresh
- Não há bloqueio de tentativas
- Mensagem de erro é genérica (bom para segurança)

## Conclusão

Sistema funcional, porém com melhorias necessárias em:
- Segurança
- Experiência do usuário
